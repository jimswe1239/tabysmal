Comment:




- To use this website, paste your tab in the tabs section. Make sure that you only enter one row of tab at a time (so don't paste a whole song, just the riff you're trying to play.) If you want to do a whole song, just do it row by row, and save the output in a text file.

- If you try to transpose too low, you'll get question marks, meaning that you're trying to play negative frets. Currently, I don't have a fix for this. It might be possible to create a workaround where the same note is added on the next lower string, five frets up, but I haven't actually tried to implement this. Also, if that string is intended to be in use, this wouldn't work anyway. So basically, don't use this app to transpose lower than the lowest fret.

- The Notes page does nothing now, but I'm planning on making a similar app which outputs the names of the notes, for people looking to play tabs on melodica or piano or whatever other instrument.

- This website will not work on tabs that are written with lines where there is no space between note names. Example below.

- IMPORTANT! Notes will be missed if they are written as a 1-digit fret played simultaneously with a 2-digit fret, where the 2-digit fret comes one hyphen before the 1-digit fret. Unfortunately, you currently must modify your tabs so that all chords are aligned on the left, not the right. Example below.


GOOD:      BAD:

e|-10-|    e|-10-|

B|-7--|    B|--7-|

GOOD:                 BAD:

A|---------1-1-1-1-|  A|------1111-|

E|-0-0-0-0---------|  E|-0000------|
